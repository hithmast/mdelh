MDE Lazy Hunter

A Python Script for MDE (Microsoft Defender for Endpoint) Investigations

What is MDE Lazy Hunter?

MDE Lazy Hunter is a Python script designed to automate security investigations using Microsoft Defender for Endpoint (MDE). It allows you to query the MDE API for various information, including:

  • Indicators of Compromise (IOCs) like file hashes and URLs
  • Device inventory details for specific devices
  • Events related to email addresses

Key Features

  • Supports multiple input types: SHA256, SHA1, MD5 hashes, IPv4 addresses, URLs, and hostnames.
  • Asynchronous processing for efficient handling of multiple queries.
  • Rate limiting to comply with MDE API restrictions.
  • CSV output for easy data analysis and sharing.
  • User-friendly prompts for error handling and interruption.
  • Custom exception handling for better error identification.

How to Use MDE Lazy Hunter

Here's a basic overview of using the script:

  1. **Install required libraries:** `aiohttp`, `dateutil`, `pytz`, `aiofiles`, `argparse` using `pip install`.
  2. **Obtain your MDE API key:** Follow the instructions in the README to extract the API key from the Microsoft Defender API Explorer.
  3. **Create a configuration file:** Create a `config.json` file with your API key.
  4. **Run the script:** Use the command line with appropriate flags depending on your investigation type:
    • For IOC processing: `python mdelh.py --iocs path/to/IOCs.txt`
    • For device inventory: `python mdelh.py --di path/to/device_names.txt`
    • For email inventory: `python mdelh.py --emails path/to/emails.txt`
  5. **Review the results:** The script will generate CSV files containing the results of your queries in the `results` folder.
